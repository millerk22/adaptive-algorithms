# Data

This directory is for datasets used by the experiment runner and notebooks.

The repository currently includes `urban.npz`. Several dataset names in `adaptive_sampling.datasets.load_dataset` expect additional local files, such as `salinasa.npz`, `pavia.npz`, `interactions.mat`, or `snps/`. Keep large or restricted datasets out of Git unless redistribution is allowed.

The synthetic and scikit-learn datasets do not require files in this directory.
