# Datasets

`adaptive_algorithms.datasets.load_dataset` returns data as `(X, labels)`, with `X` arranged as features by samples.

Built-in or generated datasets include:

- `test`
- `iris`
- `digits`
- `wine`
- `breastcancer`
- `blobs*`
- `lr`
- `lrnoisy`
- `gaussian`
- `smile`
- `outliers`

Datasets requiring local files under `data/` include:

- `urban`, `urbansub`
- `yalefaces`
- `interactions`
- `bfi`
- `salinas`
- `pavia`, `paviasub`
- `snp`
- `mnistw`

Datasets requiring optional packages include:

- UCI datasets: install `ucimlrepo`.
- Graphlearning datasets: install `graphlearning`.
